Product Offerings

WeatherTop Inc. produce a modular weather station called the WeatherTop 1000. This low cost device takes a set of meteorological readings and displays these on a simple LCD display.


Our companion web application allows users to track and analyse the weather readings from the device. Users can create individualized accounts and track weather readings by station location.

Future Enhancements

Planned future features and enhancements for WeatherTop:

  • Populate Station Dashboard with trends from historical weather reading data
  • Enhanced User Self Provisioning abilities
  • Sort Stations alphabetically
  • Data Validation during Weather Report submission
